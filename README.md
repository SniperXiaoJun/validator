# Validator

Validator基于Java Web的一款验证框架，其核心设计目的是开发迅速、代码量少、学习简单、功能强大、易扩展

---------------------------------------

Validator的特性：

* 前后端验证框架，设计精巧、使用简单
* 遵循OCP原则，强大的扩展性
* 支持SpringMVC、Struts2
* 自动生成JavaScript前端验证代码
* 将Validate与Controller分离
* 统一验证规范
* 支持自定义验证规则
